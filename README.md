# parser-tg-muiv
чат-бот, который собирает новости из выбранных Telegram-каналов и позволяет пользователю искать их по теме.
Бот использует библиотеку Telethon для парсинга сообщений и python-telegram-bot для взаимодействия с пользователем. Все данные сохраняются в SQLite, что обеспечивает быструю и автономную работу без сервера.

Основные функции:

сбор постов из заданных каналов;

хранение и обновление базы новостей;

поиск новостей по ключевым словам;

выдача результатов в удобном виде прямо в Telegram.
